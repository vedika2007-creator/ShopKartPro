🛒 ShopKart Pro – Java E-Commerce Desktop Application

📌 Overview

ShopKart Pro is a Java-based desktop e-commerce application built using Java Swing.
It simulates a real-world online shopping system with product categories, cart management, and checkout functionality.

---

🚀 Features

- 🧾 Product Catalog (Electronics, Clothing, Grocery, Sports)
- 🔍 Category Filter System
- 🛒 Add to Cart / Remove / Update Quantity
- 💳 Online & Offline Checkout Modes
- ⚡ Real-time Stock Management
- 🔄 Background Stock Sync Thread
- 📊 Dynamic UI with JTable
- 🧵 Multithreading (Payment + Stock Sync)
- ⚠️ Custom Exception Handling

---

🧩 Technologies Used

- Java (Core + OOP Concepts)
- Java Swing (GUI)
- Multithreading
- Exception Handling
- Collections (ArrayList, List)

---

🏗️ Project Structure

ShopKartPro.java
 ├── Product (Abstract Class)
 ├── Electronics, Clothing, Grocery, SportsEquipments (Inheritance)
 ├── CartItem
 ├── ShoppingCartEngine
 ├── PaymentThread
 ├── StockSyncThread
 └── GUI (JFrame + JTable)

▶️ How to Run

1. Clone the repository:

git clone https://github.com/your-username/ShopKartPro.git

2. Open in any Java IDE (IntelliJ / Eclipse / VS Code)

3. Compile and run:

javac ShopKartPro.java
java ShopKartPro
